# lesson5

A music article website built with plain HTML and CSS, based on an Adobe XD mockup.


## Structure

```
lesson5/
├── fonts/
├── images/
├── index.html
├── style.css
└── README.md
```

## Features

- Header with logo and navigation
- Full-width hero image
- Two article card grids (Popular and Recently Added)
- Dark footer with contact info and social links
- Responsive layout for mobile screens

## Usage

Open index.html directly in a browser. No build tools or dependencies required.
